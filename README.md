# Chess project

This is project for Scripting Languages at PWr

# What it does?

Project allows to play chess against chess algorithms.
It is possible to select one algorithm and play against it, you can also swap algorithm during game.

# How to install and run?

In order to install project you need to have python (3.14.5 was version that project was tested on) and installed modules from requirements.txt.

Other way to run project is to run it via docker files.
